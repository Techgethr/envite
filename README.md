## Envite – Financial Prediction Markets Mockup

This is a **Vite + TypeScript** single-page mockup for **Envite**, a conceptual platform for on-chain financial prediction markets built on the **Canton** network.

The app is purely **design and copy** – no real trading, blockchain connectivity or back-end logic.

### Run locally

```bash
npm install
npm run dev
```

Then open the printed `localhost` URL in your browser.

### Structure

- **Home / Hero**: High-level Envite value proposition and demo “Live macro sentiment” panel.
- **Markets**: Example categories for rates, inflation & growth, FX and commodities.
- **How it works**: Step-by-step description of how an institutional Canton-based platform could function.
- **Canton network**: Rationale for using Canton and an example settlement flow.
- **Use cases**: Tokenized corporate events and public policy / GDP growth scenarios.
- **FAQ**: Clarifications about the conceptual nature of the platform.

All visible copy is written in **English** for use in product pitches, decks or further UX iterations.
